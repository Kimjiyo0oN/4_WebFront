문자열 -> 숫자

        요소의 내용 또는 값을 얻어오면 기본적으로 문자열(string)으로 인식됨
        -> 덧셈 연산 시 '더하기'가 아닌 '이어쓰기'가 되는 문자 발생
    
        parseInt(문자열) :   "정수" -> 정수 ("123" -> 123)
                            "실수" -> 정수 ("3.14" -> 3)
                            (소수점 버림)
    
        parseFloat(문자열) : "정수" -> 정수 ("123" -> 123)    
                            "실수" -> 실수 ("3.14" -> 3.14)
    
        Number(문자열) :     "정수" -> 정수 ("123" -> 123)    
                            "실수" -> 실수 ("3.14" -> 3.14)

동등 비교 / 동일 비교 연산자

        동등 비교 연산자 (==, !=)
        -> 값이 true (자료형 관계 없음)


        플러스는 이어쓰기 연산이 있기 때문에 +만 문제가 있다 '10' + 5 = 105 
        나머지는 자동으로 형변환 됨  '10' - 5 = 5 ...... 
        true + 1 = 2 
        보통 true = 1
        false = 0 
        true == '1' == 1 == 1.0 => true 

        동일 비교 연산자 (===, !==)
        -> 값과 자료형이 모두 같으면 true

        1===1 => true
        true === 1 => false

문자열(string) 내장 함수

        string.indexOf("찾고 싶은 문자열")
        -> string 내에서 "찾고 싶은 문자열"의 시작 인덱스를 반환
        없으면 -1 반환

        string.lastIndexOf("찾고 싶은 문자열")
        -> 뒤에서부터 검색하여 
        string 내에서 "찾고 싶은 문자열"의 시작 인덱스를 반환
        없으면 -1 반환

        const str ="javasript";
        str.indexOf("a") => 1
        str.lastIndexOf("a") => 3 

        string.substring(시작 인덱스)
        -> 문자열에서 시작 인덱스 부터 끝까지 잘라서 반환
        ex) "Hello world".substring(2) == "llo world"

        const str = "Hello world";
        str.substring(2)
        -> 'llo world'  

        string.substring(시작 인덱스, 마지막 인덱스)
                                    (미포함)
        -> 문자열 시작 인덱스부터 마지막 인덱스(미포함) 사이를 
        잘라서 반환

        ex) "HelloWorld".substring(5,8) == "Wor"
             0123456789

        const str = "Hello World"
        str.substring(6,10);
        'Worl'
        str.substring(6,2);
        'llo '

        str.substring(str.indexOf("World"))
        str.substring(0, str.indexOf(""))

        string.split("구분자") : 문지열을 "구분자"를 기준으로 나누어 배열로 반환 

        const str = "김밥, 샌드위치, 라면, 파스타, 햄버거, 스테이크";
        const arr = str.split(", ");

숫자 관련 내장 객체 / 내장 함수

        숫자(number) 타입 리터럴 표기법 
        - 123(정수), 9.312(실수), Infinity(무한), NaN(Not a Number)
    
        숫자 관련 내장 객체 : Math

        Math.random() : 0 이상 1 미만의 난수 발생 

        (소수점 첫째 자리에 적용)
        Math.ceil(값) : 올림
        Math.floor(값) : 내림
        Math.trunc(값) : 버림
        Math.round(값) : 반올림

        숫자.toFixed(자릿수) : 숫자를 지정된 자릿수까지 반올림해서 표현

        Math.round(3.45 * 10) / 10
        3.5
        (3.45).toFixed(1) -> '3.5'

isNaN(값)

        값이 숫자가 아니면 true
        숫자면 false
